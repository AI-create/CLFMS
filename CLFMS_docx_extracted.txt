AUTH MIDDLEWARE (AUTO TOKEN PARSE)рҹ‘ү Purpose:Extract JWTAttach user to requestAvoid repeating get_current_userImplementationfrom starlette.middleware.base import BaseHTTPMiddlewarefrom jose import jwtSECRET_KEY = "secret"ALGORITHM = "HS256"class AuthMiddleware(BaseHTTPMiddleware):   async def dispatch(self, request, call_next):       token = request.headers.get("Authorization")       if token and token.startswith("Bearer "):           token = token.split(" ")[1]           try:               payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])               request.state.user = payload           except:               request.state.user = None       else:           request.state.user = None       response = await call_next(request)       return responseрҹ§  Why This MattersWithout this: вқҢ Every route decodes JWT With this: вң… request.state.user is globally availableрҹӣЎпёҸ 2.
